전체 선택자

모든 HTML 요소를 선택하여 CSS를 적용한다.

태그 선택자

특정 HTML 태그만을 선택하여 CSS를 적용한다.

아이디 선택자

HTML 태그에 id 속성을 사용하는 요소만을 찾아 CSS를 적용한다. HTML의 id 속성명과 CSS의 #id명이 동일해야 한다.

클래스 선택자

HTML 태그에 class 속성을 사용하는 요소만을 찾아 CSS를 적용한다. HTML의 class 속성명과 CSS의 .class명이 동일해야 한다.

자손/후손 선택자

  • 자손 선택자: 부모 선택자 > 자손 선택자
  • 후손 선택자: 부모 선택자 후손 선택자

자손은 부모에 종속되어 있는 태그로 1 단계 까지만 종속 관계로 묶여있는 태그이다.

후손은 부모에 종속되어 있는 태그로 n단계 까지도 종속관계로 묶어있는 태그이다.

동위 선택자

선택자1 + 선택자2
선택자1 바로 뒤에 존재하는 하나의 선택자2를 선택하기 위해 사용
+ 기호를 사용한 경우
선택자1 ~ 선택자2
선택자1 뒤에 존재하는 복수의 선택자2를 선택하기 위해 사용
~ 기호를 사용한 경우

속성 선택자

  • 기본 속성 선택자 : [속성명]
  • 일치 속성 선택자 : [속성명=속성값]
  • 공백으로 구분되는 문자열 포함 속성 선택자 : [속성명~=속성값]
  • 하이픈을 포함/미포함 하는 문자열로 시작하는 속성 선택자 : [속성명|=속성값]
  • 시작 문자열 속성 선택자 : [속성명^=속성값]
  • 종료 문자열 속성 선택자 : [속성명$=속성값]
  • 포함 속성 선택자 : [속성명*=속성값]
