년/월/일 생성하기

  • Javascript를 사용하여 현재 년 부터 ~ 1970년 까지 선택할 수 있는 선택박스를 생성한다.(select, option 태그를 사용)
  • 1 ~ 12 월 까지 선택 박스로 생성
  • 1 ~ 31 일 까지 선택 박스로 생성(선택된 열에 맞추어 28, 29, 30, 31일로 생성)
  • 선택된 년도/월에 맞추어 28, 29, 30, 31 생성할 때 change 이벤트 활용
  • Date 객체의 날짜 연산을 활용하면 각 월의 마지막 일자를 쉽게 구할 수 있다.

비밀번호/비밀번호 확인

  • 2개의 비밀번호를 받을 수 있는 사용자 입력을 생성
  • 처음 입력한 비밀번호의 값과 두번째 입력한 비밀번호의 값을 비교하여 동일한 값인지 확인
  • 동일한 값인 경우 사용자 입력 주변 테두리를 초록색으로 변경 후 입력 하단에 "정상적인 비밀번호 입니다" 메세지를 출력
  • 동일한 값이 아닌 경우 사용자 입력 주변 테두리를 붉은색으로 변경 후 입력 하단에 "비밀번호가 일치하지 않습니다" 메세지를 출력
